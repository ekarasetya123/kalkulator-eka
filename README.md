# 🧮 KalkulatorPro — Aplikasi Web Kalkulator Canggih

Aplikasi web kalkulator canggih berbasis Python Flask dengan tampilan modern.

**Dibuat oleh:** Eka Prasetya Wardana  
**Mata Kuliah:** Pengantar Pemrograman 2025/2026

---

## ✨ Fitur

### Operasi Aritmatika
- Tambah, Kurang, Kali, Bagi
- Pangkat, Akar Kuadrat, Modulus, Floor Division

### Operator Logika
- AND, OR, NOT, XOR, NAND, NOR
- Tabel kebenaran lengkap

### Transformasi Bilangan
- Konversi Basis (Decimal, Binary, Octal, Hexadecimal)
- Konversi Suhu (Celsius, Fahrenheit, Kelvin, Réaumur)
- Konversi Mata Uang (IDR ke USD, EUR, SGD, MYR, JPY, GBP, AUD, SAR)
- Faktorial (0–20)
- Deret Fibonacci (1–30)

### Fitur Tambahan
- Dark/Light Mode
- Riwayat perhitungan (History)
- Rumus + langkah penyelesaian detail
- Desain modern (Wise Design System)
- Responsif (mobile-friendly)

## 🚀 Cara Menjalankan

```bash
# Clone repository
git clone https://github.com/ekarasetya123/kalkulator-eka.git
cd kalkulator-eka

# Buat virtual environment
python -m venv venv
venv\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

# Jalankan aplikasi
python app.py
```

Buka browser: `http://127.0.0.1:5000`
